# rrr_kappa

rrr_kappa is a Matlab library for compliers DML.


## Usage

```
master.m
```
For the empirical illustration of DML-RRR in the context of effect of 401(k) on assets, we obtain data from the replication files for 
Alexandre Belloni, Victor Chernozhukov, Ivan Fernández-Val, and Christian Hansen. Program evaluation and causal inference with high-dimensional data. Econometrica, 85(1):233?298,3042017.
